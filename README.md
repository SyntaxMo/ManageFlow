# MangeFlow

MangeFlow is a Next.js and Supabase project management workspace for game development teams.

## Phase 1 Included

- Next.js App Router with TypeScript
- Tailwind CSS color palette and global styles
- Public landing page and auth page shells
- Protected dashboard layout shell
- Reusable UI components
- Supabase browser/server clients
- Middleware route guard scaffold
- Role and permission helper scaffold
- Initial Supabase schema/RLS starter file

## Setup

1. Copy `.env.example` to `.env.local`.
2. Add Supabase project URL and anon key.
3. Install dependencies with `pnpm install`.
4. Start development with `pnpm dev`.

The first dashboard uses sample data until the Supabase tables are connected in later phases.
