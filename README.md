# 🎮 ManageFlow

ManageFlow is a role-based management system designed for game development teams.

It helps admins, team leads, project managers, employees, and interns manage users, teams, attendance, daily reports, meeting requests, and project progress from one platform.

## 💡 Why ManageFlow Is Important

Game development teams often use different tools for attendance, reporting, communication, and task tracking. ManageFlow brings these workflows together in one system.

It helps improve:

- 👥 Team organization
- 💬 Communication between different roles
- 🎓 Intern supervision
- 📈 Daily progress tracking
- ✅ User approval and team assignment
- 🕒 Attendance and reporting workflows

## 🛠️ Tech Stack

- ⚡ Next.js
- 🔷 TypeScript
- 🎨 Tailwind CSS
- 🗄️ Supabase

## 🗺️ Development Roadmap

The project is being developed in phases to ensure a scalable and maintainable architecture.

- **Phase 1:** 🚀 Project setup, authentication, database design, and user registration.
- **Phase 2:** 👤 Role-based dashboards, user approval system, profiles, and settings.
- **Phase 3:** 📝 Daily reports, attendance, meeting requests, and intern workflows.
- **Phase 4:** 👥 Team management, project management, and role hierarchy.
- **Phase 5:** ✅ Task management, project timelines, milestones, and progress tracking.
- **Phase 6:** 📁 File management, notifications, messaging, and calendar integration.
- **Phase 7:** 📊 Analytics, reports, performance dashboards, exports, and final system optimization.
